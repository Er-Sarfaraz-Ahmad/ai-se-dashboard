#  AI SEO Audit Dashboard

An AI-powered SEO Audit Dashboard built using React, Vite, Tailwind CSS, and Google PageSpeed API to analyze website SEO performance, mobile friendliness, security, and optimization insights.

## 📌 Features

✅ Website SEO Analysis  
✅ Performance Score Tracking  
✅ Mobile Friendliness Score  
✅ Security Analysis  
✅ Smart SEO Summary Dashboard  
✅ Website Health Indicator  
✅ Export SEO Report (.txt)  
✅ Responsive UI (Mobile + Desktop)

---

## 🛠️ Tech Stack

**Frontend**
- React.js
- Vite
- Tailwind CSS

**API**
- Google PageSpeed Insights API

## ⚙️ Installation

Clone the repository:

```bash
git clone YOUR_GITHUB_REPO_LINK
```

Go to project folder:

```bash
cd ai-seo-dashboard
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```

---

## 🔑 Environment Variables

Create a `.env` file in root folder and add:

```env
VITE_GOOGLE_API_KEY=your_api_key
```
